# Scala SMTP Mail Client

A [Scala](http://scala-lang.org/) [SMTP](https://en.wikipedia.org/wiki/Simple_Mail_Transfer_Protocol) mailer
built on top of the [JavaMail API](https://javamail.java.net/nonav/docs/api/).
